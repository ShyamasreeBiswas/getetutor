Overview

Subastral Inc Admin Control Panel is a php coded web page allowing you to display the html content of files on your server in a standard browser. Using standard browser View Source functions php is not displayed as it is processed by the server prior to the page being delivered to the browser.

Subastral Inc Admin Control Panel runs on your web server. To run Admin Control Panel your server must support the php scripting language. Most commercial web servers support php. For information on php and how to check if it is available on your server goto What is php?

Instructions for Use - Admin Control Panel

Go to Subastral Inc Admin Control Panel and login with your user id and password. At the left side of the page click on Admin Panel to nominate Allowed Directories. By default the only Allowed Directory is the directory in which the Admin Panel is located. The Allowed Directory can be viewed using the Admin Panel. If you only wish to view files in the Admin Panel directory then no configuration is required.

Trouble Shooting

Take Care when modifying and uploading you file to ensure the php coding is not corrupted. If your Admin Control Panel does not run when you submit check the form action. When you view index.php in your test editor the form tag is:

<form action="<?php echo "$_SERVER['PHP_SELF']"; ?>" method="post">

The action is php code telling the form to load itself. When this is run on the server the action becomes the Admin Control Panel file name. If the form action is missing the php code may have been corrupted when you edited or uploaded the file. To overcome this you can manually set the form action to the file name i.e.:

<form action="index.php" method="post">
